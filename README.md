# Chapter 4 Sample Code

Here, you'll find a sampler of sample code!

Each of the items has lots of added information in its own README.

1. An autoencoder example, in the folder called [autoencoder](autoencoder/).  In this case, I'm using only the Pandas from the Cat - Dog - Panda dataset.  I also only used Convolution layers, you can also add other kinds of layers.
1. An RNN example, in the folder called [RNN](RNN/).  In this case, the code is mostly from a text, but I am still working on a version that is more like the code I've developed earlier in the course.
1. A Word Embeddings example, in the folder called [Word Embeddings](word_embeddings/).  This uses a bag of words approach with a word-to-vector embedding, trained on a problem that classifies IMDB reviews as positive or negative, based on the number of stars attached to the review.  The model learns a substantial amount of the meaning of words from this starting point.
1. In my main Github account, I have two examples of Reinforcement Learning using a Deep Q Network. One solves the [cartpole problem](https://github.com/TBSDrJ/Torch_RL_Cartpole) and does good job of it, following a tutorial.  The other tries to learn to play [2048](https://github.com/TBSDrJ/2048-Torch), but the results are pretty bad, and I can't get it to improve, though I've tried a wide variety of modifications.
1. See what your convolution filters are finding, and what they train to! In the folder [visualization](visualization/).
